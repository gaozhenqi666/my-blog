## 1. 产品概述
一个炫酷的暗色系个人博客页面，用于自我介绍和展示。通过动态效果和特殊符号占位符营造科技感氛围，吸引访客注意力。

目标用户：个人开发者、设计师、创意工作者，希望通过独特的方式展示个人形象和专业技能。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览博客内容、查看动效 |

### 2.2 功能模块
博客页面包含以下核心模块：
1. **主页**：英雄区域、个人介绍、技能展示、动态背景效果
2. **关于页面**：详细个人信息、经历介绍、特殊符号占位内容
3. **作品集页面**：项目展示、技术栈展示、动态卡片效果

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页 | 英雄区域 | 展示炫酷的暗色背景动画，使用#¥%*&等符号作为装饰元素 |
| 主页 | 个人介绍 | 显示姓名、职位、简短描述，配合打字机动画效果 |
| 主页 | 技能展示 | 以卡片形式展示技术技能，带有悬停动效 |
| 主页 | 动态背景 | 粒子效果、符号飘动、渐变色彩变化 |
| 关于页面 | 个人信息 | 详细介绍，使用特殊符号作为段落分隔符 |
| 关于页面 | 经历时间线 | 垂直时间轴展示经历，带有滚动触发动画 |
| 作品集页面 | 项目卡片 | 网格布局的项目展示，悬停时有3D翻转效果 |
| 作品集页面 | 技术标签 | 动态标签云展示掌握的技术 |

1. 访问主页 → 2. 浏览英雄区域动画 → 3. 查看个人介绍 → 4. 浏览技能展示 → 5. 进入关于页面 → 6. 查看作品集
